# Шаблон Gruvbox Air

Цей шаблон створено під впливом [Gruvbox-Rainbow](./gruvbox-rainbow.md)

![Скріншот шаблона Gruvbox Air](/presets/img/gruvbox-air.png)

### Передумови

- Встановлений та увімкнений шрифт [Nerd Font](https://www.nerdfonts.com/) у вашому терміналі

### Налаштування

```sh
starship preset gruvbox-air -o ~/.config/starship.toml
```
