## Plain Text Symbols Preset

此预设将所有组件的符号改为纯文本。 适合没有 Unicode 的环境。

![Screenshot of Plain Text Symbols preset](/presets/img/plain-text-symbols.png)

### 配置

```sh
starship preset plain-text-symbols > ~/.config/starship.toml
```
