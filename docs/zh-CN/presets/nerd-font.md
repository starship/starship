# Nerd 字体符号预设

此预设使用 Nerd Font 的符号显示所有组件。

![Nerd Font 符号预设截图](/presets/img/nerd-font-symbols.png)

### 前置要求

- 安装一种 [Nerd Font](https://www.nerdfonts.com/) 并在您的终端启用 (示例使用的是 Fira Code 字体)。

### 配置

```sh
starship preset nerd-font-symbols -o ~/.config/starship.toml
```
