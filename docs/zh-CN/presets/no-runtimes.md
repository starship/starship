# No Runtime Versions Preset

此预设隐藏语言运行时版本。 如果你在容器或虚拟环境中使用，这个适合你！

![Screenshot of Hide Runtime Versions preset](/presets/img/no-runtime-versions.png)

### 配置

```sh
starship preset no-runtime-versions > ~/.config/starship.toml
```
