# Catppuccin Powerline 预设

该预设是在 [Gruvbox Rainbow](./gruvbox-rainbow.md) 的基础上进行最小修改，并使用了 [Catppuccin](https://github.com/catppuccin/catppuccin) 主题配色方案。

![Catppuccin Powerline 预设截图](/presets/img/catppuccin-powerline.png)

### 前置要求

- 在你的终端中已安装并启用 [Nerd Font](https://www.nerdfonts.com/)

### 配置

```sh
starship preset catppuccin-powerline -o ~/.config/starship.toml
```

默认情况下，此预设使用 Catppuccin 的 Mocha 风格，但你可以通过修改 `palette` 的值来指定任意风格：

- `catppuccin_mocha`
- `catppuccin_frappe`
- `catppuccin_macchiato`
- `catppuccin_latte`
