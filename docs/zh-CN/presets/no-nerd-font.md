# No Nerd Fonts Preset

此预设谨慎的使用了来自emoji和powerline的符号。

这意味着即使没有安装Nerd Font，您也应该看到所有的模块符号。

在未来的starship release中，此预设将成为默认预设。

### 配置

```sh
starship preset no-nerd-font -o ~/.config/starship.toml
```
