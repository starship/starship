## 일반 텍스트 기호 프리셋

이 프리셋은 각 모듈의 기호를 일반 텍스트로 변경합니다. 유니코드에 액세스할 수 없는 경우에 좋습니다.

![일반 텍스트 기호 프리셋 스크린샷](/presets/img/plain-text-symbols.png)

### 설정

```sh
starship preset plain-text-symbols -o ~/.config/starship.toml
```
