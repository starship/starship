# No Nerd Fonts プリセット

このプリセットは使用するシンボルを絵文字や powerline のセットに制限します。

つまり Nerd Font をインストールしなくても、すべてのモジュールのシンボルを正しく表示できます。

このプリセットは、将来の Starship のリリースで既定のプリセットになります。

### 設定

```sh
starship preset no-nerd-font -o ~/.config/starship.toml
```
