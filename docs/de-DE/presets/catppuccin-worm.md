# Pastel Worm Preset

Dieses Preset ist eine minimal modifizierte Version von [Catppuccin Powerline](./cattpuccin-powerline.md), die die [Catppuccin](https://github.com/catppuccin/catppuccin) Themen-Palette benutzt.

![Screenshot der Catppuccin Worm Auswahl](/presets/img/catppuccin-worm.png)

### Voraussetzungen

- Ein [Nerd Font](https://www.nerdfonts.com/) muss installiert und im Terminal aktiviert sein.

### Konfiguration

```sh
starship preset catppuccin-worm -o ~/.config/starship.toml
```

Dieses Preset nutzt in der Voreinstellung das Mocha Bukett von Catppucin. Aber man kann jedes andere Bukett einstellen, indem der Parameter `palette` geändert wird:

- `catppuccin_mocha`
- `catppuccin_frappe`
- `catppuccin_macchiato`
- `catppuccin_latte`
