### Voraussetzungen

Gehe zur [Chocolatey's Installations-Seite](https://chocolatey.org/install) und folge den Anweisungen um Chocolatey zu installieren.

```powershell

```

## [termux](https://termux.com)

### Voraussetzungen


