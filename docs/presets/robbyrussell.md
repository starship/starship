# robbyrussell Preset

This preset emulates the look and behavior of [Oh My Zsh's robbyrussell](https://github.com/ohmyzsh/ohmyzsh/wiki/themes#robbyrussell).

![Screenshot of robbyrussell preset](/presets/img/robbyrussell.jpg)

### Configuration

```sh
starship preset robbyrussell -o ~/.config/starship.toml
```
