# Earth Night Preset

A warm, earthy take on the [Tokyo Night](./tokyo-night) layout — same structure, re-themed with brown, tan, and olive tones.

![Screenshot of Earth Night preset](/presets/img/earth-night.png)

### Prerequisites

- A [Nerd Font](https://www.nerdfonts.com/) installed and enabled in your terminal

### Configuration

```sh
starship preset earth-night -o ~/.config/starship.toml
```
