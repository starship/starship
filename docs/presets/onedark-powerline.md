# OneDark Powerline Preset

This preset is a minimally modified version of [Gruvbox Rainbow](./gruvbox-rainbow.md) using [Atom](https://github.com/atom/atom)'s OneDark theme palette.

![Screenshot of OneDark Powerline preset](/presets/img/onedark-powerline.png)

### Prerequisites

- A [Nerd Font](https://www.nerdfonts.com/) installed and enabled in your terminal

### Configuration

```sh
starship preset onedark-powerline -o ~/.config/starship.toml
```
