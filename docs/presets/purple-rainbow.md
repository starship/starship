# Purple Rainbow

This preset is inspired by [Gruvbox Rainbow Preset](https://starship.rs/presets/gruvbox-rainbow) and [Tokyo Night Preset](https://starship.rs/presets/tokyo-night)

A smooth purple-to-blue gradient prompt inspired by Gruvbox Rainbow and Tokyo Night.

![Screenshot of Purple Rainbow preset](/presets/img/purple-rainbow.png)

### Prerequisites

- A [Nerd Font](https://www.nerdfonts.com/) installed and enabled in your terminal (the example uses Caskaydia Cove Nerd Font)

### Configuration

```sh
starship preset purple-rainbow -o ~/.config/starship.toml
```
