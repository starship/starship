# Bracketed Segments Preset

這個預設樣式使所有模組使用括號片段內的格式顯示，而非使用 Starship 預設（"via", "on", etc.）。

![括號片段預設樣式的截圖](/presets/img/bracketed-segments.png)

### 設定

```sh
starship preset bracketed-segments -o ~/.config/starship.toml
```
