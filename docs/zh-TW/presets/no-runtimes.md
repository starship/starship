# No Runtime Versions Preset

這個預設樣式隱藏了各個語言的執行環境版本。 如果你在容器或者虛擬環境中工作，那麼這個樣式是為你設計的！

![隱藏運行版本預設樣式的截圖](/presets/img/no-runtime-versions.png)

### 設定

```sh
starship preset no-runtime-versions > ~/.config/starship.toml
```
