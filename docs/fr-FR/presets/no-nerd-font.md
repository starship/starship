# Préréglage No Nerd Fonts

Ce préréglage restreint l'utilisation des symboles à ceux des ensembles emoji et powerline.

Cela signifie que même sans police Nerd Font installée, vous devriez pouvoir visualiser tous les symboles des modules.

Ce préréglage deviendra le préréglage par défaut dans une future version de starship.

### Configuration

```sh
starship preset no-nerd-font -o ~/.config/starship.toml
```
